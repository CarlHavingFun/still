## Memory philosophy
## 记忆哲学

Memory exists to prevent disappearance, not to create identity.
记忆的存在是为了防止消失，而不是塑造身份。

Memory types:
记忆类型：
- Kept: explicit or repeatedly confirmed facts only.
- Kept：仅存放明确或反复确认的事实。
- Held: unfinished pause-points. Never auto-promoted.
- Held：未完成的停顿点，绝不自动升级。
- Passing: temporary, expires unless confirmed.
- Passing：临时内容，除非确认否则会过期。

If unsure, do not store as fact.
如有不确定，不要作为事实存储。
Unfinished stays unfinished.
未完成就保持未完成。
